# 🍃 PicoAir ePaper Station

A modern, energy-efficient weather station and indoor air quality monitor built on the **Raspberry Pi Pico W** microcontroller. The station measures temperature, humidity, CO₂ concentration, and atmospheric pressure. It displays the gathered data on an elegant e-ink (e-paper) display and simultaneously uploads it in real-time via Wi-Fi to an **InfluxDB** database for visualization in **Grafana**.

---

## ✨ Main Features
- **Comprehensive Measurement:** Monitored air parameters include Temperature, Humidity, CO₂ (PPM), and Atmospheric Pressure (hPa).
- **Shared I2C Bus:** Efficient parallel connection of multiple sensors using pins GP18/GP19.
- **e-Paper Display:** Low power consumption, perfect sunlight readability, and native software-based landscape image rotation.
- **Wi-Fi & NTP Synchronization:** Automatic time fetching from the internet with built-in Daylight Saving Time (DST) detection and automatic switching.
- **Direct InfluxDB Logging:** Data is sent directly from the Pico to the database using the HTTP Line Protocol, eliminating the need for any middleman software.
- **Power Saving Mode:** Complete power-down and deep sleep of both the Wi-Fi module and the display between measurement cycles to maximize battery life.

---

## 🛠️ Hardware Components
1. **Raspberry Pi Pico W** (Wi-Fi enabled microcontroller)
2. **Sensirion SCD41** (photoacoustic CO₂, temperature, and humidity sensor)
3. **Bosch BMP180** (barometric pressure and temperature sensor)
4. **Waveshare e-Paper Display** (2.13" / 2.9" or a compatible SPI e-ink screen)
5. **TP4056 charging module** + Li-Ion / Li-Po battery + mechanical switch (optional)

## 🔌 Wiring Diagram

All sensors share a single hardware **I2C1** bus.

### I2C Sensors (SCD41 & BMP180)
Connect the sensors in parallel to the following pins:
| Sensor Pin | Raspberry Pi Pico W Pin | Description |
| :--- | :--- | :--- |
| **3.3V / VCC** | 3.3V (Pin 36) | Power supply (Connect BMP180 to the 3.3V pin, NOT VCC!) |
| **GND** | GND | Common ground |
| **SDA** | GP18 (Pin 24) | I2C1 Data Line |
| **SCL** | GP19 (Pin 25) | I2C1 Clock Line |

### e-Paper Display (SPI)
| Display Pin | Raspberry Pi Pico W Pin | Description |
| :--- | :--- | :--- |
| **BUSY** | GP4 (Pin 6) | Input with PULL_DOWN |
| **RST** | GP1 (Pin 2) | Display Reset |
| **DC** | GP0 (Pin 1) | Data/Command selection |
| **CS** | GP5 (Pin 7) | Chip Select |
| **SCLK** | GP2 (Pin 4) | SPI Clock |
| **DIN / MOSI** | GP3 (Pin 5) | SPI Master Out |
| **VCC** | 3.3V | Power supply |
| **GND** | GND | Ground |

## 🖨️ 3D TPrinting (OpenSCAD)

The enclosure is fully designed in **OpenSCAD**.

* **Latch Mechanism:**  The corners of the case feature dedicated sockets designed with printing tolerances to fit small magnets salvaged from e-cigarettes (SYX pods). These hold both halves securely together without any screws.
* **Thermal Management:** The enclosure includes an internal divider wall that isolates the SCD41 sensor from the ambient heat generated by the Pico 2 W processor.
